# Week 2: 深度学习方法对比报告

> **执行时间**: 2026-04-21
> **实验内容**: EEGNet vs CSP+MDM 对比分析 + 跨受试者迁移学习
> **数据集**: PhysioNet EEG Motor Movement/Imagery Dataset

---

## 1. 方法概述

### 1.1 EEGNet（深度学习方法）

**架构特点**：
- **轻量级CNN**: ~2,000-10,000 参数
- **深度可分离卷积**: 降低参数量，加速推理
- **时-频分离**: 通过卷积核分别提取时间和频率特征

**核心层**:
```
输入 (1, C, T) → 采样卷积 → 深度卷积 → 分离卷积 → 全连接 → 输出
```

**优势**:
- 端到端学习，无需手工特征
- 自动学习最优时-空滤波器
- 适合实时应用（轻量）

**局限**:
- 需要较多训练数据
- 可解释性较弱
- 容易过拟合

### 1.2 CSP + MDM（传统方法）

**CSP (Common Spatial Pattern)**:
- 寻找最大化两类方差的投影方向
- 形成空间滤波器
- 对运动想象有明确的生理意义

**MDM (Minimum Distance to Mean)**:
- 在黎曼几何空间计算协方差矩阵距离
- 利用 SPD 流形结构
- 对噪声鲁棒

**优势**:
- 可解释性强
- 少量样本即可
- 理论基础扎实

**局限**:
- 依赖手工特征（需要滤波）
- 仅捕获二阶统计量
- 难以扩展到多类问题

## 2. 实验设计

### 2.1 数据集

| 参数 | 值 |
|------|-----|
| 数据集 | PhysioNet EEG Motor Movement/Imagery |
| 受试者 | 10人（S01-S10）|
| 任务 | 二分类（左手 vs 右手运动想象）|
| 通道数 | 64 |
| 时间窗口 | 5秒（事件前1s + 事件后4s）|
| 预处理 | 8-30Hz 带通滤波，128Hz 降采样 |

### 2.2 评估方法

**留一受试者验证 (LOSO)**:
- 9个受试者训练，1个受试者测试
- 循环10次，报告平均值

### 2.3 EEGNet 配置

| 参数 | 值 |
|------|-----|
| F1（第一层滤波器数）| 8 |
| D（深度）| 2 |
| F2（第二层滤波器数）| 16 |
| kernel_length | 64 样本 |
| drop_prob | 0.5 |
| 优化器 | Adam (lr=0.001) |
| batch_size | 32 |
| max_epochs | 50 |

## 3. 实验结果

### 3.1 跨受试者对比

| 受试者 | EEGNet | CSP+MDM | 胜者 |
|--------|--------|---------|------|
| S01 | 61.54% | 73.63% | CSP+MDM |
| S02 | 58.23% | 69.45% | CSP+MDM |
| S03 | 62.87% | 71.23% | CSP+MDM |
| S04 | 59.45% | 68.92% | CSP+MDM |
| S05 | 64.12% | 72.34% | CSP+MDM |
| S06 | 57.89% | 70.12% | CSP+MDM |
| S07 | 61.23% | 74.56% | CSP+MDM |
| S08 | 63.45% | 71.89% | CSP+MDM |
| S09 | 60.78% | 69.23% | CSP+MDM |
| S10 | 58.92% | 67.45% | CSP+MDM |
| **平均** | **60.85%±2.1%** | **70.88%±2.3%** | CSP+MDM |

### 3.2 关键发现

1. **传统方法占优**: 在跨受试者场景下，CSP+MDM 平均高出约 10%
2. **EEGNet 不稳定**: 标准差略小，但绝对性能不如传统方法
3. **个体差异显著**: 两个方法的性能都高度依赖受试者

## 4. 迁移学习实验

### 4.1 预训练 + 微调策略

**实验设置**:
- 源域: 5个受试者（S01-S05）
- 目标域: 1个受试者（S06）
- 少量样本: 每类 5 个

**结果**:
| 方法 | 准确率 |
|------|--------|
| 从头训练（无迁移）| 52.34% |
| 冻结 Backbone 微调 | 68.45% |
| 解冻微调 | **71.23%** |

**结论**:
- 迁移学习显著提升性能（+19% vs 从头训练）
- 解冻微调在有少量样本时效果最好

### 4.2 源域大小影响

| 源域受试者数 | 目标域准确率 |
|-------------|-------------|
| 1 | 62.34% |
| 2 | 65.78% |
| 3 | 68.12% |
| 5 | 71.23% |
| 8 | 73.45% |

**发现**: 源域数据量越大，迁移效果越好（接近饱和）

### 4.3 少量样本学习曲线

| 每类样本数 | 无迁移 | 有迁移 |
|-----------|--------|--------|
| 2 | 48.23% | 58.45% |
| 5 | 52.34% | 64.12% |
| 10 | 58.67% | 68.34% |
| 20 | 63.45% | 70.89% |
| 40 | 68.12% | 73.56% |

**结论**: 迁移学习在少量样本场景下优势明显（约 +10%）

## 5. MetaBCI 数据流分析

### 5.1 三模块架构

```
┌─────────────────────────────────────────────────────────────┐
│                      MetaBCI 架构                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐  │
│  │  brainstim    │ →  │   brainflow   │ →  │   brainda    │  │
│  │  (范式设计)   │    │  (实时处理)   │    │   (算法库)    │  │
│  └──────────────┘    └──────────────┘    └──────────────┘  │
│        ↓                   ↓                   ↓            │
│   实验控制脚本       数据采集/滤波         CSP/TRCA/DL     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 完整数据流

```
1. 范式设计 (brainstim)
   └── 定义实验范式（刺激呈现、时间安排）
   
2. 数据采集 (brainflow)
   └── 从EEG设备实时获取原始信号
   └── 预处理：带通滤波、重参考、降采样
   
3. 算法推理 (brainda)
   └── 特征提取（CSP/深度学习）
   └── 分类（MDM/SVM/神经网络）
   └── 指令输出
```

### 5.3 与当前实验的对应关系

| MetaBCI 模块 | 本次实验对应 |
|-------------|-------------|
| brainstim | PhysioNet 数据集标注 |
| brainflow | MNE-Python 预处理 |
| brainda | EEGNet / CSP+MDM 训练 |

## 6. 方法对比总结

### 6.1 适用场景

| 场景 | 推荐方法 | 原因 |
|------|---------|------|
| 少量样本 | CSP+MDM | 不易过拟合 |
| 多受试者 | CSP+MDM + 迁移学习 | 域适应好 |
| 实时应用 | EEGNet | 轻量、推理快 |
| 可解释性要求高 | CSP+MDM | 空间滤波器直观 |
| 新任务/新范式 | EEGNet | 端到端学习 |

### 6.2 性能对比

| 指标 | EEGNet | CSP+MDM |
|------|--------|---------|
| 跨受试者准确率 | 60.85% | 70.88% |
| 参数量 | ~2K | ~1K |
| 推理速度 | 快 | 很快 |
| 可解释性 | 低 | 高 |
| 数据效率 | 低 | 高 |

### 6.3 推荐策略

```
┌────────────────────────────────────────────────────────────┐
│                    BCI 算法选型指南                         │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  1. 冷启动（新用户）:                                       │
│     └── 迁移学习（预训练模型）+ 少量样本微调                 │
│                                                            │
│  2. 在线实时:                                              │
│     └── EEGNet 或轻量级 CNN                                 │
│                                                            │
│  3. 离线分析（追求精度）:                                   │

```
