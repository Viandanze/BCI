## ✅ 本周完成清单

### 1. 集成学习框架（核心产出）
**文件：** `src/training/ensemble.py`（600+行）

| 策略 | 类 | 说明 |
| --- | --- | --- |
| Voting | `VotingEnsemble` | 硬投票/软投票，支持异构模型组合 |
| Stacking | `StackingEnsemble` | 元学习器（Logistic/SVM/Ridge），交叉验证生成元特征 |
| Weighted | `WeightedEnsemble` | 基于验证性能的softmax权重 |
| Bagging | `BaggingEnsemble` | Bootstrap聚合，n_bagging个模型 |
| Adaptive | `AdaptiveEnsemble` | 门控网络动态权重 |

### 2. 实时推理Pipeline（核心产出）
**文件：** `src/inference/pipeline.py`（600+行）

| 组件 | 类 | 功能 |
| --- | --- | --- |
| 数据缓冲 | `StreamingBuffer` | 滑动窗口，支持步长提取 |
| 预处理 | `PreprocessWorker` | 异步带通滤波+归一化 |
| 推理引擎 | `InferenceEngine` | PyTorch模型加载+批推理 |
| 后处理 | `PostProcessor` | 平滑+投票+置信度阈值 |
| 完整流水线 | `RealTimePipeline` | 协调所有组件 |
| 模拟数据流 | `MockDataStream` | 模拟MI EEG信号 |

### 3. 迁移学习框架
**文件：** `src/training/transfer.py` + `scripts/transfer_learn.py`

| 方法 | 模式 | 适用场景 |
| --- | --- | --- |
| DANN | 域对抗 | 源/目标域差异大 |
| CORAL | 相关对齐 | 无标签目标域 |
| 逐层解冻 | Fine-tune | 小数据集 |
| 差分学习率 | LR decay | 预训练模型微调 |
| 对比学习 | Contrastive | 增强泛化 |

### 4. 高级模型
**文件：** `src/models/advanced.py`

- **ShallowConvNet**：浅层卷积，专注频域特征
- **EEG-Conformer**：Transformer架构，跨数据集领先
- **TCN**：时序卷积网络，适合长时程依赖

### 5. 可视化模块
**文件：** `src/visualization/visualizer.py`

8种可视化 + 综合Dashboard：
- 训练曲线、混淆矩阵、EEG拓扑图、频谱图等

### 6. 跨受试者评估
**文件：** `src/evaluation/cross_subject.py`

- LOSO（Leave-One-Subject-Out）评估
- 域偏移分析
- 受试者相似度矩阵
