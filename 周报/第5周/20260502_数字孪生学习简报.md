## 核心概念速览

### MQTT vs OPC UA 选型（2026最新）

| 维度 | MQTT | OPC UA |
|------|------|--------|
| 定位 | 物联网轻量级传输 | 工业信息建模框架 |
| 资源消耗 | 几十KB客户端库 | 需要网关/工控机 |
| 数据语义 | 需额外构建元数据 | 内置丰富语义模型 |
| 适用场景 | 传感器采集、边缘层 | 复杂设备关系集成 |

**融合架构（主流方案）**：边缘层OPC UA → 云端MQTT

### QoS等级选择

| 等级 | 语义 | 适用场景 |
|------|------|----------|
| QoS 0 | 至多一次 | 高频传感器数据（如温湿度） |
| QoS 1 | 至少一次 | 控制指令（如开灯），需去重 |
| QoS 2 | 恰好一次 | 关键告警，开销大 |

### 关键机制

- **Retained Message**：Broker保存最后一条消息，新订阅者立即获取最新值
- **LWT (Last Will & Testament)**：设备异常断开时自动发布，用于检测离线状态

### 步骤3：MQTT数据发布服务

**sensor_publisher.py**

```python
import paho.mqtt.client as mqtt
import json
import time
from datetime import datetime
from sensor_dht11 import DHT11
from sensor_bmp280 import BMP280

# MQTT配置
MQTT_BROKER = "localhost"
MQTT_PORT = 1883
CLIENT_ID = "edge-node-rpi"
TOPIC_PREFIX = "home/sensors/"

# 初始化传感器
dht11 = DHT11(pin=4)
bmp280 = BMP280()

# MQTT客户端
client = mqtt.Client(CLIENT_ID)
client.connect(MQTT_BROKER, MQTT_PORT, 60)

def publish_sensor_data():
    """采集并发布所有传感器数据"""
    timestamp = datetime.now().isoformat()
    
    # DHT11数据
    temp, hum = dht11.read()
    if temp is not None:
        payload = {
            "device_id": "dht11_01",
            "temperature": round(temp, 2),
            "humidity": round(hum, 2),
            "timestamp": timestamp
        }
        client.publish(TOPIC_PREFIX + "dht11", json.dumps(payload), qos=1)
    
    # BMP280数据
    temp, press, alt = bmp280.read()
    if temp is not None:
        payload = {
            "device_id": "bmp280_01", 
            "temperature": round(temp, 2),
            "pressure": round(press, 2),
            "altitude": round(alt, 2),
            "timestamp": timestamp
        }
        client.publish(TOPIC_PREFIX + "bmp280", json.dumps(payload), qos=1)

# 主循环
while True:
    try:
        publish_sensor_data()
        time.sleep(5)  # 5秒采集一次
    except Exception as e:
        print(f"Error: {e}")
        time.sleep(10)
```

### 步骤4：本地告警与GPIO联动

**local_alert.py**

```python
import paho.mqtt.client as mqtt
import RPi.GPIO as GPIO
import json

ALERT_PIN = 18
GPIO.setmode(GPIO.BCM)
GPIO.setup(ALERT_PIN, GPIO.OUT)

def on_message(client, userdata, msg):
    try:
        data = json.loads(msg.payload.decode())
        
        # 温度超阈值告警
        if "temperature" in data and data["temperature"] > 30:
            print(f"温度过高: {data['temperature']}°C")
            GPIO.output(ALERT_PIN, GPIO.HIGH)
        else:
            GPIO.output(ALERT_PIN, GPIO.LOW)
    except Exception as e:
        print(f"解析失败: {e}")

client = mqtt.Client("alert-engine")
client.on_message = on_message
client.connect("localhost", 1883, 60)
client.subscribe("home/sensors/#")  # 订阅所有传感器话题
client.loop_forever()
```

### 步骤5：Web可视化仪表盘

**web_dashboard.py**

```python
from flask import Flask, render_template
import paho.mqtt.client as mqtt
import sqlite3
import json
from datetime import datetime

app = Flask(__name__)

# SQLite存储
conn = sqlite3.connect('sensor_data.db', check_same_thread=False)
cursor = conn.cursor()
cursor.execute('''CREATE TABLE IF NOT EXISTS readings 
    (device TEXT, data TEXT, timestamp DATETIME)''')

# MQTT订阅存储
def on_message(client, userdata, msg):
    data = json.loads(msg.payload.decode())
    device = msg.topic.split('/')[-1]
    cursor.execute("INSERT INTO readings VALUES (?, ?, ?)",
                   (device, msg.payload.decode(), datetime.now()))
    conn.commit()

mqtt_client = mqtt.Client("web-subscriber")
mqtt_client.on_message = on_message
mqtt_client.connect("localhost", 1883)
mqtt_client.subscribe("home/sensors/#")
mqtt_client.loop_start()

@app.route('/')
def index():
    return render_template('dashboard.html')

@app.route('/api/data/<device>')
def get_data(device):
    cursor.execute(f"SELECT timestamp, data FROM readings WHERE device=? ORDER BY timestamp DESC LIMIT 100", (device,))
    rows = cursor.fetchall()
    return json.dumps(rows)

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```

## 关键提示

- 用户已有DHT11、BMP280、HC-SR04传感器，可以直接开始硬件实验
- 建议先用 `mosquitto_sub` 验证MQTT通断，再接传感器
- GPIO操作注意电压，3.3V逻辑
- 传感器接线参考：
  - DHT11: DATA→GPIO4, VCC→3.3V, GND→GND
  - BMP280: SDA→GPIO2, SCL→GPIO3, VCC→3.3V, GND→GND
