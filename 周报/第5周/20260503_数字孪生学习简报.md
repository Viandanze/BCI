## 核心概念速览

### 为什么数字孪生需要卡尔曼滤波？

传感器数据天然存在噪声和不确定性，直接使用会导致：
- 仪表盘数值跳动
- 控制系统响应抖动
- 预测模型精度下降

**卡尔曼滤波的价值**：在噪声环境中对动态系统状态进行最优估计，是数字孪生"数据同化"的核心算法。

### 一维卡尔曼滤波四步公式

| 步骤 | 公式 | 说明 |
|------|------|------|
| **预测** | `x_pred = x_est + Q` | 基于上一时刻最优估计预测当前状态 |
| **增益计算** | `K = P / (P + R)` | 平衡预测信任度与测量信任度 |
| **状态更新** | `x_est = x_pred + K(z - x_pred)` | 融合预测值与测量值 |
| **协方差更新** | `P = (1-K)P` | 更新估计不确定性 |

### Q/R参数工程含义

| 参数 | 含义 | 调大效果 | 调小效果 |
|------|------|----------|----------|
| Q（过程噪声） | 模型不确定性 | 滤波曲线更平滑，响应变慢 | 跟踪更灵敏，可能引入噪声 |
| R（测量噪声） | 传感器精度 | 信任测量值，更灵敏 | 信任预测值，更平滑 |

## 实战任务：卡尔曼滤波 + Three.js 实时可视化

### 任务1：卡尔曼滤波数据融合（Python）

**目标**：对DHT11和BMP280的温度数据进行融合去噪

**kalman_filter.py**

```python
import numpy as np
import json
import time
import paho.mqtt.client as mqtt

class KalmanFilter1D:
    """一维卡尔曼滤波器"""
    def __init__(self, process_noise=0.1, measurement_noise=1.0):
        self.q = process_noise   # 过程噪声Q
        self.r = measurement_noise  # 测量噪声R
        self.x = None           # 估计值
        self.p = 1.0            # 估计协方差
        self.k = 0              # 卡尔曼增益
    
    def filter(self, measurement):
        if self.x is None:
            self.x = measurement  # 初始化
            return self.x
        
        # 预测
        x_pred = self.x
        p_pred = self.p + self.q
        
        # 更新
        self.k = p_pred / (p_pred + self.r)
        self.x = x_pred + self.k * (measurement - x_pred)
        self.p = (1 - self.k) * p_pred
        
        return self.x

# MQTT配置
BROKER = "localhost"
PORT = 1883
TOPIC_INPUT = "home/sensors/#"
TOPIC_OUTPUT = "home/sensors/fused"

# 创建滤波器（温度/湿度/气压各一个）
kf_temp = KalmanFilter1D(process_noise=0.05, measurement_noise=0.5)
kf_hum = KalmanFilter1D(process_noise=0.1, measurement_noise=1.0)
kf_press = KalmanFilter1D(process_noise=0.01, measurement_noise=0.1)

fused_data = {"temperature": None, "humidity": None, "pressure": None}

def on_message(client, userdata, msg):
    try:
        payload = json.loads(msg.payload.decode())
        topic = msg.topic
        
        # 根据topic选择滤波器
        if "dht11" in topic and "temperature" in payload:
            fused_data["temperature"] = kf_temp.filter(payload["temperature"])
        elif "dht11" in topic and "humidity" in payload:
            fused_data["humidity"] = kf_hum.filter(payload["humidity"])
        elif "bmp280" in topic and "pressure" in payload:
            fused_data["pressure"] = kf_press.filter(payload["pressure"])
        
        # 发布融合结果（当所有数据都就绪）
        if all(v is not None for v in fused_data.values()):
            client.publish(TOPIC_OUTPUT, json.dumps(fused_data))
            print(f"融合结果: 温度={fused_data['temperature']:.2f}°C, "
                  f"湿度={fused_data['humidity']:.2f}%, "
                  f"气压={fused_data['pressure']:.2f}hPa")
    except Exception as e:
        print(f"处理错误: {e}")

# 主循环
client = mqtt.Client("kalman-fusion")
client.on_message = on_message
client.connect(BROKER, PORT, 60)
client.subscribe(TOPIC_INPUT)
print("开始卡尔曼滤波融合...")
client.loop_forever()
```

**server.js（WebSocket转发MQTT数据）**

```javascript
const express = require('express');
const { WebSocketServer } = require('ws');
const mqtt = require('mqtt');
const app = express();

app.use(express.static('public'));
app.listen(3000);

// MQTT订阅
const mqttClient = mqtt.connect('mqtt://localhost:1883');
mqttClient.subscribe('home/sensors/fused');

// WebSocket广播
const wss = new WebSocketServer({ port: 8080 });
wss.on('connection', ws => {
    console.log('Web客户端连接');
});

mqttClient.on('message', (topic, message) => {
    const data = JSON.parse(message.toString());
    // 广播给所有WebSocket客户端
    wss.clients.forEach(client => {
        if (client.readyState === 1) {
            client.send(JSON.stringify(data));
        }
    });
});
```

        import * as THREE from 'https://unpkg.com/three@0.160.0/build/three.module.js';
        
        // 场景设置

        scene.background = new THREE.Color(0x1a1a2e);
        
        const camera = new THREE.PerspectiveCamera(75, window.innerWidth/window.innerHeight, 0.1, 1000);
        camera.position.set(0, 2, 5);
        
        const renderer = new THREE.WebGLRenderer({ antialias: true });
        renderer.setSize(window.innerWidth, window.innerHeight);
        document.body.appendChild(renderer.domElement);

            bar.position.set((i - 1) * 1.5, 0.5, 0);
            scene.add(bar);
            bars.push(bar);

        // 添加灯光
        const ambientLight = new THREE.AmbientLight(0xffffff, 0.5);
