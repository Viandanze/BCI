# LeetCode 797 - 所有可能的路径

## 题目信息

| 项目 | 内容 |
|------|------|
| 题号 | 797 |
| 标题 | All Paths From Source to Target |
| 难度 | 中等 (Medium) |
| 链接 | https://leetcode.com/problems/all-paths-from-source-to-target/ |

## 题目描述

给定一个有向无环图（DAG），包含 `n` 个节点（编号 0 到 n-1），和一个二维整数数组 `graph`，其中 `graph[i]` 表示节点 `i` 的所有出边邻居。

返回从节点 **0** 到节点 **n-1** 的所有可能路径。

## 解题思路

### 核心算法：深度优先搜索 (DFS) + 回溯

由于要求**所有**路径，使用 DFS 遍历所有从 0 到 n-1 的路径。

**算法步骤**：
1. 从节点 0 开始 DFS，将当前节点加入路径
2. 如果当前节点是目标节点 n-1，保存当前路径
3. 否则，遍历当前节点的所有出边，递归访问邻居
4. 回溯：移除当前节点，尝试其他分支

### 代码实现要点

```cpp
void dfs(node):
    path.add(node)
    if node == target:
        result.add(path.copy)
    else:
        for neighbor in graph[node]:
            dfs(neighbor)
    path.remove(node)  // 回溯
```

## 复杂度分析

### 时间复杂度：O(N × 2^N)

- **N** = 节点数量
- 在最坏情况下（DAG为完全有向无环图），从 0 到 n-1 的路径数为指数级
- 每条路径长度为 O(N)，复制路径需要 O(N)
- 总计：O(N × 2^N)

### 空间复杂度：O(N + H)

- **O(N)**：递归栈深度，最长路径为 N
- **O(H)**：存储结果的空间，H 为所有路径的总节点数
- 最坏情况下 H = N × 2^N（每条路径长度 N，数量 2^N）

## 优化方向

| 优化策略 | 说明 |
|---------|------|
| **剪枝** | 如果已知某些节点无法到达终点，提前跳过 |
| **记忆化** | DP 记录从某节点到终点的路径数（如果只求数量） |
| **BFS + 栈** | 用迭代方式替代递归，避免栈溢出 |
| **路径去重** | 如果需要唯一路径，使用 set 去重 |
