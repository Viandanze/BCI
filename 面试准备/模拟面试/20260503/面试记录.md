## 一、算法环节

### 题目1：LeetCode 200 - 岛屿数量

```cpp
class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        if (grid.empty() || grid[0].empty()) return 0;
        
        int count = 0;
        int m = grid.size(), n = grid[0].size();
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    count++;
                    bfs(grid, i, j);
                }
            }
        }
        return count;
    }
    
private:
    void bfs(vector<vector<char>>& grid, int i, int j) {
        queue<pair<int, int>> q;
        q.push({i, j});
        grid[i][j] = '0';
        
        int dirs[4][2] = {{-1,0}, {1,0}, {0,-1}, {0,1}};
        
        while (!q.empty()) {
            auto [x, y] = q.front();
            q.pop();
            
            for (auto& d : dirs) {
                int nx = x + d[0], ny = y + d[1];
                if (nx >= 0 && nx < grid.size() && 
                    ny >= 0 && ny < grid[0].size() && 
                    grid[nx][ny] == '1') {
                    grid[nx][ny] = '0';
                    q.push({nx, ny});
                }
            }
        }
    }
};
```

- **时间复杂度**：O(m×n)
- **空间复杂度**：O(min(m×n, B))

### 题目2：LeetCode 114 - 二叉树展开为链表

```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        TreeNode* curr = root;
        while (curr) {
            if (curr->left) {
                TreeNode* pred = curr->left;
                while (pred->right) pred = pred->right;
                pred->right = curr->right;
                curr->right = curr->left;
                curr->left = nullptr;
            }
            curr = curr->right;
        }
    }
};
```

- **核心思路**：利用前驱节点作为桥梁，空间O(1)

```
┌─────────────────────────────────────────────────┐
│              World Partition 动态加载架构        │
├─────────────────────────────────────────────────┤
│  [玩家位置] → [距离计算] → [加载队列]           │
│      ↓           ↓           ↓                 │
│  [LOD决策]   [优先级排序] [Streaming Handler]  │
│                     ↓                          │
│              [Grid Cell Manager]               │
└─────────────────────────────────────────────────┘
```

**核心参数**：
| 参数 | 值 | 说明 |
|------|-----|------|
| LoadRadius | 5000 | 基础加载半径 |
| UnloadRadius | 8000 | 卸载半径 |
| GridSize | 10000 | 每Cell尺寸 |

**优化策略**：
1. 预加载（根据移动速度预测）
2. 优先级队列（视线方向权重 +30%）
3. 异步流式（StreamableManager）
4. 分帧加载（每帧最多2个Cell）
